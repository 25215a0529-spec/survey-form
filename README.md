# Survey Form

A responsive survey form built with HTML as part of the **Responsive Web Design Certification** from FreeCodeCamp.

## 📌 About

This project demonstrates the fundamentals of HTML forms, including:

- Text, email, and number inputs
- HTML5 form validation
- Dropdown menus
- Radio buttons
- Checkboxes
- Textarea for additional feedback
- Submit button

## 🚀 Technologies Used

- HTML5
